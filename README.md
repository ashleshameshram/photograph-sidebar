## 📷 Photography Portal — Sidebar UI
A mini front-end project built to practice and showcase HTML & CSS skills through a clean, dark-themed photography portal interface.
The project features a sleek responsive sidebar with navigation links — Gallery, Shortcut, Exhibits, Events, Store, Contact, and Feedback — each paired with icons for a polished look. The layout is complemented by a full-screen background image and social media links at the bottom, giving it the feel of a real-world photography website.

## 🎯 Purpose
This project was built purely as a CSS & HTML practice exercise, focusing on:

- Sidebar layout and styling
- Dark theme UI design
- Icon integration
- Typography and spacing
- Overall visual composition
